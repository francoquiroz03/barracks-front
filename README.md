## Requirements

* Node 16
* Git
* Docker

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/francoquiroz03/barracks-front.git
cd barracks-front
```

```bash
docker-compose build
docker-compose up -d
```

## Steps for access

* Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
